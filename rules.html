L'Escalier - Règles du jeu

But du jeu

Le but du jeu est de gagner le maximum de plis possible en pronostiquant le nombre de plis que vous allez gagner.

Règles de base

  • Le jeu se joue à deux joueurs.
  • Chaque joueur doit pronostiquer le nombre de plis qu'il va gagner.
  • Le joueur qui gagne le plus de plis possible remporte la partie.
  • Si le nombre de plis gagnés est égal au pronostic, le joueur gagne un bonus de 5 points.
  • Si le nombre de plis gagnés est inférieur au pronostic, le joueur perd 5 points.
  • Si le nombre de plis gagnés est supérieur au pronostic, le joueur perd 5 points pour chaque pli en trop.
